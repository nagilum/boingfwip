Boingfwip

Boingfwip is a simple API that lets you do a request and get all the info back as a JSON.

What's With the Name?

I'm terrible at naming my projects and went through several rounds in my head with different stuff. Ended up with Boingfwip, because reasons..

A phrase used by the character Ted and his band on the popular television series "scrubs". This is a short hand version of "I'm rubber and you're glue" implying someone's insult bounces off of you, creating a "boing" sound and is thrown back through the air making a "fwip" sound. Boing fwip is often accompanied by hand movements implying said action.

person 1: You're a jerkface
person 2: Boing fwip!
jerk face (formerly known as person 1): Oh snap

Source: urbandictionary.com

The Process

The API endpoint is https://europe-west2-boingfwip-api.cloudfunctions.net/query and should answer to almost any HTTP method, but at least the normal ones.

You do a request towards the endpoint and you'll get a JSON in return. It's that simple. Everything you send in, from headers, to query parameters, to body content should be reflected in the JSON you get back.

Limitations

The only limitations of the API is that the size of the submitted body payload. The limit is set to 10 MB.

Response Body Explanation

  • request contains the your IP, HTTP method as well as info about the HTTP version used.
  • headers contains the headers you submitted.
  • query contains any query parameters you submit.
  • cookies contains the cookies you submitted.
  • body contains the body payload, both raw and analyzed (if possible).
